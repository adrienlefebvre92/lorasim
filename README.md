# Lorasim

## Installation

1. You first need to clone the repository locally

2. Then the virtual env has to be created from the project folder
```
python3 -m venv lora-env  # do not change the name
```

3. Activatate the virtual env
```
source lora-env/bin/activate 
```

3. Then install the required dependancies in the venv
```
pip3 install -r requirements.txt
```

4. You can use it !
```
./loraDir.py 10 10 3 2000 2
```
